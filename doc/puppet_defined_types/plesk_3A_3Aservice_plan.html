Summary

A short summary of the purpose of this defined type.

Overview

Create service plan with desired specifications

All params have their details below: docs.plesk.com/en-US/obsidian/cli-linux/using-command-line-utilities/service_plan-hosting-plans.38365/ Note that those with dashes, have been changed to the same but with underscores.

Examples:

plesk::service_plan { 'PlanName':
  ensure              => present,
  disk_space          => "100M",
  php_served_by_nginx => "true",
  overuse             => "block"
}
